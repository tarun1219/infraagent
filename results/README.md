# Results Directory

This directory contains experiment results and generated figures.

## Structure

```
results/
├── experiment_results.json   # Pre-computed results for all 8 models × 5 conditions
├── figures/                  # Generated paper figures (PDF + PNG)
│   ├── fig01.pdf / fig01.png  # Overall accuracy by condition
│   ├── fig02.pdf / fig02.png  # Self-correction improvement by round
│   ├── ...
│   └── fig14.pdf / fig14.png  # RAG retrieval quality
├── raw/                      # Per-run JSONL files (generated by run_experiments.py)
│   └── {model}_{condition}_run{N}.jsonl
├── README.md                 # This file
└── STATISTICS.md             # Statistical significance summary
```

## experiment_results.json

The main results file contains the following top-level keys:

| Key | Description |
|-----|-------------|
| `main_results` | Per-condition accuracy across all 8 models |
| `by_difficulty` | Accuracy broken down by L1–L5 difficulty level |
| `security_delta` | Security pass rate change: one-shot → sc+rag+3r |
| `round_improvement` | Per-round accuracy improvement curves |
| `rag_ablation` | RAG contribution: one-shot vs one-shot+rag |
| `multi_run_statistics` | Mean ± 95% CI, Cohen's d, p-values (5 runs) |
| `terraform_deploy_localstack` | Validation vs deployment gap (LocalStack) |
| `k8s_dry_run_server_full` | Server-side validation gap by difficulty |
| `layer_ablation` | Accuracy drop when each validation layer is removed |
| `rag_retrieval_quality` | Precision@5, Recall@5, MRR, NDCG@5 by difficulty |

## Reproducing Results

```bash
# Regenerate from pre-computed data (no model inference required)
python scripts/simulate_results.py
python scripts/generate_figures.py

# Full reproduction with live model inference
bash scripts/reproduce_paper.sh --live
```

## Raw Results Format

Each `results/raw/{model}_{condition}_run{N}.jsonl` file contains one JSON object per task:

```json
{"task_id": "k8s-001", "functional": true, "security": true, "round": 1, "difficulty": 2, "type": "kubernetes"}
{"task_id": "tf-045",  "functional": true, "security": false, "round": 3, "difficulty": 4, "type": "terraform"}
```
